# Met Jonah

Een simpele Nederlandse familie-pagina met activiteiten in de buurt van Berlicum (Pijlkruid) voor onze kleine Jonah.

- Live: https://metjonah.nl
- Stack: één HTML-bestand, geen framework, geen backend
- Weer: Open-Meteo (gratis, geen API key)
- Update-frequentie: elke 3 dagen via een Cowork scheduled task
